----------------------------------
ABOUT THE PROJECT
----------------------------------
This is a Pokémon Daycare Management System made with Java and MySQL.
It lets daycare staff add, view, update, and remove Pokémon from a daycare.
You can also calculate how much each Pokémon owes for their stay.

----------------------------------
WHAT YOU NEED
----------------------------------
- Java (JDK 17 or newer)
- IntelliJ IDEA (or another Java program)
- MySQL Community Server + MySQL Workbench
- MySQL Connector/J library (added in IntelliJ)

----------------------------------
SETUP STEPS
----------------------------------
1. Open MySQL Workbench.
2. Run the file called: pokemon_daycare.sql
3. Open this project in IntelliJ.
4. Run the file: DaycareGUI.java

----------------------------------
WHEN YOU RUN THE PROGRAM
----------------------------------
You’ll be asked for:
- MySQL server: (example: localhost:3306)
- Database name: pokemon_daycare
- Username: (example: root)
- Password: (your MySQL password)

Then the program will open a window with buttons to:
- Add Pokémon
- Update Pokémon
- Remove Pokémon
- View all Pokémon
- Calculate fee
